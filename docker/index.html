イメージ確認

docker images

コンテナ確認

docker ps # 停止中も含めて確認 docker ps -a

コンテナ全停止

docker stop $(docker ps -aq) # 停止中も含めて確認 docker rm $(docker ps -aq)

コンテナ全削除

docker rm $(docker ps -aq)

イメージ全削除

docker rmi $(docker images -q)

全削除

docker system prune

$ docker prune は、container, image, network, volumeなど
